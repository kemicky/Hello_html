
/* 
       How to change comments color
       ----------------------------
Go to settings -> type "editor.tokenColorCustomizations " -> Edit setting.json file
Then add the following line of code to the file :
"editor.tokenColorCustomizations": {
        
    "comments": "#FF0000"
   }
   And save the changes. 

   location of settings.json file : Library -> Application Support -> Users ->Code -> User -> settings.json
